A cinematic, professional-level birthday wish website for Tanu Singh's 20th birthday on March 25, 2026.

## ✨ Features

🌟 **Cinematic Design**
- Animated starfield background with twinkling effects
- Gradient overlays and beautiful UI
- Smooth page transitions and scroll animations
- Premium typography and spacing

⏳ **Live Countdown Timer**
- Real-time countdown to birthday
- Days, Hours, Minutes, Seconds display

- Integrated audio player with auto-play
- Control volume and playback

- 6 personalized wish cards for Tanu
- Radha Ji blessings
- Hardwork & achievement recognition
- Mysterious personality appreciation
- Motivational messages

🎊 **Interactive Elements**
- Click anywhere to trigger confetti bursts
- Floating animations
- Smooth hover effects
- Particle system with physics

📱 **Mobile-First Design**
- Desktop optimized
- Tablet friendly
- Phone-first responsive design
- "Scroll for more" indicator
- Perfect viewing on mobile devices
