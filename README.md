# birthday-wish28
A cinematic birthday wish website for Tanu Singh's 18th birthday with countdown timer, special wishes, confetti animations, and song.

---

## 🚀 How to Use

1. Open the website in any modern web browser
2. Enjoy the cinematic birthday experience
3. **Click anywhere** on the page to trigger confetti animations
4. **Scroll down** to see all special wishes
5. **Play the music** using the audio player
6. Share the link with Tanu! 💕

## 🎨 Customization

To personalize this website:

- **Change birthday date:** Edit line in `script.js` (look for `2026-03-25`)
- **Change songs:** Replace `song.mp3` with your desired audio file
- **Change colors:** Modify gradient values in `style.css`
- **Edit wishes:** Update wish card text in `index.html`
- **Change name:** Replace "Tanu Singh" in `index.html`
